# 🧮 Math Expression Solver

A terminal-based Python app that solves and analyzes mathematical expressions step by step — built with SymPy and Rich.

---

## ✨ Features

- **Expression Analyzer** — Simplify, differentiate, integrate, and find roots of any expression
- **Limit Solver** — Compute limits as x approaches any value including infinity
- **Matrix Operations** — Determinant, inverse, and eigenvalues/eigenvectors
- **System of Equations** — Solve 2 or 3 variable systems instantly

## 🚀 Getting Started

### Prerequisites
- Python 3.8+

### Installation

1. Clone the repo
```bash
   git clone https://github.com/perez-larsey/math-solver.git
   cd math-solver
```

2. Install dependencies
```bash
   pip install sympy rich
```

3. Run the app
```bash
   python solver.py
```

## 🧪 Example Inputs

| Feature | Input Example |
|---|---|
| Expression | `x**2 + 3*x - 5` |
| Limit | `sin(x)/x` as x → `0` |
| Matrix | 2x2 or 3x3 matrix entries |
| System | `x + y - 10` and `2*x - y - 4` |
